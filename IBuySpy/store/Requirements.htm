Setup Requirements

  

Before you install IBuySpy Store, you should be sure your machine meets the following setup requirements:

  • Windows 2000 or Windows XP (Server or Professional)
  • SQL Server or MSDE 7.0 SP2 (or higher)
  • .NET Framework SDK or Visual Studio .NET
  • Microsoft Internet Explorer 5.5 (or higher)

Notes:  Be sure to activate the SQL Agent before running the IBuySpy Store setup steps. This will make sure that the nightly scheduled job to clean out abandoned shopping carts runs properly.
